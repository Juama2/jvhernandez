GitHub

Github es un portal creado para alojar el código de las aplicaciones de cualquier desarrollador, y que fue comprada por Microsoft en junio del 2018.

Neocities

Neocities es un servicio de alojamiento web comercial para páginas estáticas. Ofrece 1 GB de espacio de almacenamiento para sitios gratuitos y sin secuencias de comandos del lado.

Backup

En todos los ámbitos de la tecnología es necesario contar con una o varias copias de seguridad o backups de todos nuestros trabajos.

Color Hexadecimal

El RGB es la mezcla aditiva de tres luces de color Rojo, Verde y Azul (Red, Green y Blue), y es el estándar para medir el color en dispositivos tecnológicos.

CSS

El CSS es un lenguaje que vino a dar vida al diseño y desarrollo web, facilitando en gran medida el poder asignar diferentes estilos a objetos de manera más o menos sencilla.

Drop Down menú

Cuando en un menú haces clic o te pones sobre él y se despliega una persiana con varias opciones, eso es un Drop-Down menú.

Headings

Los encabezados HTML, comúnmente conocidos como H1, H2, H3, etc. son unas etiquetas que se utilizan para organizarycategorizar jerárquicamente los contenidos de una página web.

Error 404

Es un código de estado HTTP que nos está diciendo que la página que queremos abrir no existe o no está disponible.

Iframe

Un Iframe es un recurso de código HTML que nos permite insertar en una página un documento externo, una página web, etc., para que forme parte del contenido de nuestra página.

Javascript

El JavaScript (JS) es un lenguaje de programación que se utiliza para hacer cosas complejas en una página web.

Landing page

Una landing page es una página creada únicamente con un fin: la captación de leads o potenciales clientes.

Masonry layout

El estilo «Masonry» es un tipo de maquetación web consistente en aprovechar los espacios en blanco que quedan entre los diferentes objetos de una página.

URL

La URL es una dirección que se pone en nuestro navegador web para poder ver cualquier recurso que exista en Internet.

Z-Index

‘Z-Index’ es un propiedad de código CSS que determina en qué orden colocamos varios elementos que están puestos unos encima de otros.

Slider

El Slider o carrusel es un recurso de diseño web que consiste en hacer que varias imágenes (u otros elementos) se vayan desplazando horizontalmente de manera automática.

Address

Elemento de bloque, elemento contenedor. El elemento de dirección se utiliza ocasionalmente; permite a los visitantes de un sitio Web ponerse en contacto con el autor de la página.

API

Interfaz de Programación de Aplicaciones. Una serie de rutinas usadas por una aplicación para gestionar generalmente servicios de bajo nivel, realizados por un sistema.

Article

Representa el concepto que tenga de artículo: una sección de contenido independiente que puede ser un artículo de un periódico, un mensaje de un foro o una entrada de un blog.
